# Cryptography exercise

## Functions implementation

Write the following functions:

a) A function that creates an ECDSA-P256 (secp256r1) key and saves it in PEM
format.  
b) A function that reads a PEM key and uses it to sign a message. The signature
should be returned in Base64URL format.  
c) A function that reads a PEM key and uses it to verify the signature of a
message generated by the second (b) function.
