# Cryptography exercise

## Functions implementation

Write the following functions:

a) A function that creates an ECDSA-P256 (secp256r1) key and saves it in PEM
format.  
b) A function that reads a PEM key and uses it to sign a message. The signature
should be returned in Base64URL format.  
c) A function that reads a PEM key and uses it to verify the signature of a
message generated by the second (b) function.

## Data structures implementation

Write a structure for storing one or more statuses (boolean true/false). The
structure should use the []byte type to store the statuses, where each
individual bit represents one state (true or false).

The structure should support:

- Setting an individual existing state.
- Creating a new state within the list.
- Encoding the entire list into a gzipped, Base64-encoded string.  
encodedList: (return base64(gzip(byteArray)))
